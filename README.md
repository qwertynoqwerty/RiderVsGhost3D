# RiderVsGhost3D

![Unity](https://img.shields.io/badge/Unity-2020.3%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

Мини-гонка с режимом «призрак» в духе гоночных аркад: два заезда — в первом вы проезжаете трассу, во втором против вас едет ваш «призрак», повторяющий предыдущий заезд.
Пройдите трассу быстрее своего прошлого результата!

---

## 🎯 Основные возможности

- **2 заезда**:
  - Первый — запись траектории игрока
  - Второй — гонка против призрака
- **Обратный отсчёт** перед стартом
- **Таймер** заезда
- **Финишная система** с экраном результата
- **Призрак** повторяет движение игрока, включая повороты
