**RiderVsGhost** — тестовое задание, выполненное на Unity.  
Игрок проходит трассу дважды: в первом заезде игра записывает движение машины, а во втором запускает призрака, который повторяет записанную траекторию.

Проект сделан с упором на управление машиной, запись движения, повтор траектории и базовую логику гоночного заезда.

---

## 🎮 Основные возможности

- 🚗 **Два заезда** — первый заезд записывает движение игрока, второй запускает гонку против призрака.
- 👻 **Призрак игрока** — повторяет записанный путь машины.
- 🏁 **Таймер гонки** — показывает текущее время прохождения трассы.
- ⏱ **Обратный отсчёт** — запускается перед началом заезда.
- 🔁 **Повтор заезда** — после финиша можно начать гонку заново.
- 🏠 **Возврат в меню** — после второго заезда можно вернуться на главный экран.

---

## 🕹 Как играть

1. Запустите игру.
2. В главном меню нажмите **Start**.
3. Дождитесь окончания обратного отсчёта.
4. Пройдите трассу — первый заезд будет записан.
5. После финиша начнётся второй заезд с призраком.
6. Попробуйте обогнать собственный записанный результат.
7. После второго финиша выберите действие:
   - **Повторить** — начать заново.
   - **В меню** — вернуться на главный экран.

---

## 🚀 Запуск проекта

1. Склонируйте репозиторий:

```bash
git clone https://github.com/destrosez/RiderVsGhost.git
```

2. Откройте проект в **Unity 6.0+**.
3. Дождитесь импорта ассетов.
4. Откройте основную сцену проекта.
5. Нажмите **Play**.
